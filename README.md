# 📈 Pro 股息記錄器

專業級的配息追蹤分析工具，協助你掌握股票配息狀況。

## ✨ 特色功能

- **即時配息記錄** - 快速新增配息資料
- **智能統計分析** - 總額、平均、筆數一覽
- **時間軸圖表** - 按月配息趨勢視覺化
- **分類標籤管理** - 穩定股/高股息/潛力股
- **日期範圍篩選** - 自定義時間軸查詢
- **全文搜尋** - 股票名稱或備註即時檢索
- **CSV 匯入匯出** - 資料備份與批量導入
- **響應式設計** - PC/平板/手機皆完美支援
- **離線可用** - 資料存在瀏覽器，不依賴伺服器

## 🚀 快速開始

### 使用方法

1. **新增配息記錄**
   - 輸入股票名稱/代碼
   - 選擇配息日期
   - 輸入配息金額
   - 選擇分類（穩定股/高股息/潛力股）
   - 填寫備註（選填）

2. **查詢與篩選**
   - 設定開始/結束日期
   - 選擇特定股票
   - 使用搜尋框快速查找

3. **資料管理**
   - 匯出 CSV 備份資料
   - 匯入 CSV 批量新增
   - 單筆刪除紀錄
